First of all thanks to visit my website. I am developing this website after learning about basic HTML and CSS from this youtube playlist by The Net Ninja. I love how beautifully he teaches.

I will divide my learning broadly into the followings

  • First of all we create a skeleton of a website using HTML.
  • After that, we style the site by linking the page with CSS.
  • Classes are very useful in hooking some particular items in the HTML to style them.
  • We can also use id for similar goal but that's majorly used while making radio buttons
  • Inheritence is an important quality while styling.
  • inline-block are best kind of blocks. They make elements inline and we can also use margins and padding as if they're block elements.
  • pseudo classes are some cool stuff. We can style elements using this for when the user focus or hover over certain elements.
  • Using media queries we can make the page responsive. We can preset different styles for different page sizes.
